# TikTok-style SPA

This repository contains a small React + Vite single page app scaffold that mimics a TikTok-style landing page with hero video, product cards, animations (Framer Motion), and Tailwind CSS.

How to run locally

1. Install dependencies

```bash
npm install
```

2. Run dev server

```bash
npm run dev
```

3. Open http://localhost:5173

Notes

- The project uses placeholder assets located in /public. Replace `placeholder.mp4` and thumbnail images in `/public` with your own assets.
- Deploy to Vercel or Netlify by connecting this repository.
